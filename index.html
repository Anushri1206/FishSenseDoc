FishSense iOS Application

This project simplifies fish measurement and monitoring during catch-and-release operations. Instead of manually measuring and identifying each fish, we’re developing an iOS app that uses the device’s camera and LiDAR to detect fish, identify head and tail positions, and calculate length automatically. With machine learning, the app will also identify species and streamline data collection for marine or citizen scientists.

Key Features

  • Accurate Fish Length Detection – Uses the iOS device’s camera and LiDAR to identify head and tail positions and calculate length.
  • Fish Species Detection – Applies machine learning to recognize different fish species in real-time.
  • Session Management & Cloud Upload – Organizes catch sessions and uploads collected data to the cloud for easy access and analysis.

Repository Organization

We are using two main repositories for this project: Follow the README in both the repositories for replicating it

  • iOS Swift App – Contains the core iOS application that handles fish detection, measurement, and session management.
    This repository includes the Swift code for real-time fish detection using the iPhone's camera and LiDAR, with session management features.

    Please check out the README in this repository for detailed setup and usage instructions.

  • fishsense.rs – A supporting library with Rust-based fish length estimation logic.
    This Rust library provides the core algorithms for estimating fish length from sensor data, designed to be efficient and accurate.

    Please check out the README in this repository for detailed build and integration steps.

Fish Species Classification Model

We use a fish classification model based on the Fishial model, and exported directly from TorchScript to ONNX format for lightweight deployment. This allows us to run the model efficiently on iOS devices using ONNX Runtime.

The ONNX model is publicly hosted on Hugging Face for reproducibility and integration:

ONNX Fish Classifier – Hugging Face

This model enables real-time, on-device fish species recognition within our iOS app.
