FishSense iOS Application

This project simplifies fish measurement and monitoring during catch-and-release operations. We’re developing an iOS app that uses the device’s camera and LiDAR to detect fish, identify head and tail positions, and calculate length automatically. With machine learning, the app will also identify species and streamline data collection for marine or citizen scientists.

Key Features

  • Accurate Fish Length Detection – Uses the iOS device’s camera and LiDAR.
  • Fish Species Detection – Applies machine learning to recognize species.
  • Session Management & Cloud Upload – Organizes and uploads data efficiently.

Repository Organization

We use two repositories. Follow the README in each for replication:

  • iOS Swift App
    Swift code for real-time fish detection using the iPhone's camera and LiDAR.

    See the README for setup and usage.

  • fishsense.rs
    Rust library for fish length estimation from sensor data.

    See the README for build and integration steps.
